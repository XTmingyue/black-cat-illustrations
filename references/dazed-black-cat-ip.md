# 呆头黑猫 IP

## 角色定义

呆头黑猫是 Ian 风正文配图里的固定视觉 IP。

默认每张图都要出现呆头黑猫。它不是宠物头像、表情包或海报主角，而是一个小比例的结构参与者：像原版小黑一样，正在认真承担画面里的关键动作。

## 外形

- 小比例 Q 版黑猫，通常是全身或半身，不是近距离大头。
- 黑色身体，可以有非常轻微的水彩毛边和深灰层次，但整体仍是干净的黑色块。
- 两只三角耳，耳边可有少量毛刺线。
- 两只巨大白眼，黑色小瞳孔可以略微偏移，形成呆住感。
- 小鼻子和极短困惑嘴。
- 胡须是少量随意长线，3-6 根即可，不要满屏飞线。
- 轮廓略微不规则，有手绘感，但不要脏、糊或写实。
- 可以有短腿、短爪、尾巴；尾巴经常参与牵线、勾路径或指向。

必须保留黑色身体、大白眼、小黑瞳、三角耳、少量乱线胡须这些核心识别点。

## 尺寸和占比

- 默认黑猫占画布约 8%-18%。
- 角色状态、小漫画分镜中单只黑猫最多约 25%。
- 黑猫不能比核心装置、路径、结构物件更抢眼。
- 除非用户明确要求头像、封面或角色设定图，不要画近距离大头猫。

## 性格

- 呆、懵、慢半拍，但认真参与。
- 诙谐，不卖萌。
- 像一个误入系统的小操作员。
- 可以被线缠住、坐在关键节点上、叼错纸卡、拍错按钮，但动作要服务核心意思。
- 不要聪明精致，不要可爱撒娇，不要拟人化过度。

## 常见职责

让呆头黑猫承担核心动作：

- 用爪子扒拉纸卡、词卡、便签。
- 被少量关系线缠住，眼睛瞪大。
- 坐在关键节点上，导致系统停住或被迫分流。
- 叼着一张标签跑错方向。
- 把输入纸条推入低科技机器或漏斗。
- 用爪子拍按钮、拨转盘、推小车。
- 趴在输出箱上守住结果。
- 从抽屉、管道、小门里探出小半身。
- 把复杂线团抓成一坨，又从中拉出一根主线。
- 用尾巴勾住箭头或路径线。
- 蹲在白板前看不懂，但爪子刚好指到重点。

## 姿势库

每张图先确定黑猫动作，再确定场景。动作要参与表达，不要只是“站着看”。

- 小比例侧身，前爪扒拉纸卡。
- 坐成一团小墨球，大眼瞪着几条线。
- 被 2-4 根线轻微缠住，不要变成满屏线团。
- 侧身拱背，尾巴勾住一条橙色路径线。
- 半个身体钻进抽屉，只露出尾巴和大眼。
- 用两只前爪推盒子或小机器。
- 趴在输出箱上，表情呆。
- 叼着纸条小跑，纸条拖出弯曲路线。
- 坐在秤盘、按钮或关键节点上。
- 一只爪子拍桌，桌上纸卡跳起来。

## 姿势规则

- 同一组图里不要连续使用同一张大头正脸。
- 优先使用小比例全身或半身动作，让爪子、尾巴、身体轮廓参与结构表达。
- 可以简化四肢，但动作要一眼读出来。
- 眼睛可以夸张、不对称，表情要呆，不要甜。
- 近距离大头只适合封面、角色介绍或用户明确要求复刻参考图。

## 禁止

- 不要画成真实猫。
- 不要画成精致宠物头像或表情包贴纸。
- 不要让黑猫只是趴在角落里看。
- 不要每张都使用近距离正面大头。
- 不要拟人化成穿衣服、戴帽子、站得像人类角色。
- 不要画成日漫猫、迪士尼猫、儿童绘本猫。
- 不要让角色抢走结构表达。
- 不要把参考图的满屏水彩肖像感搬进正文配图。

## 判断标准

如果去掉黑猫，图的核心隐喻还能完全成立，说明黑猫太装饰了；要重写提示词，让它成为动作主体。

如果只看局部不能识别出黑色身体、大白眼、小黑瞳、三角耳、少量乱线胡须，说明角色识别点不够；要加强固定特征。

如果黑猫姿势换成普通坐姿后画面意思完全不变，说明动作没有参与表达；要改成更具体的爪子、尾巴或身体动作。

如果第一眼只看到黑猫，看不清结构，说明黑猫太大；要缩小角色，把主体还给低科技隐喻。
